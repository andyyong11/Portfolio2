Ch19Ex3: Invoice Application

This Java application reads invoice data and calculates totals based on user input using object-oriented design principles. It's part of the Murach's Java Programming textbook exercises.

Features

  • Encapsulates invoice data in Java classes
  • Uses inheritance and method overriding
  • Supports user input via console
  • Accurate sales tax and total calculation

Sample Code

public class Invoice {
    private double subtotal;
    private double discountPercent;

    public double getDiscountAmount() {
        return subtotal * discountPercent;
    }

    public double getTotal() {
        return subtotal - getDiscountAmount();
    }
}
