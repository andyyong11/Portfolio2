Ch19Ex3: Product Manager Application

A JavaFX desktop application that manages product inventory with full CRUD (Create, Read, Update, Delete) functionality using a SQLite database. This application demonstrates advanced object-oriented programming concepts and database integration.

Features

  • Modern JavaFX GUI with intuitive user controls
  • Add, search, update, and delete product records
  • Integrates with SQLite database for data persistence
  • Input validation and error handling
  • MVC architecture with separation of concerns

Sample Code

public class ProductManagerApp extends Application {
    // JavaFX components
    private TextField codeField;
    private TextField descriptionField;
    private TextField priceField;
    private ListView productList;
    
    // Database operations
    private void addProduct() {
        Product product = new Product(
            codeField.getText(),
            descriptionField.getText(),
            Double.parseDouble(priceField.getText())
        );
        ProductDB.add(product);
        populateProductList();
    }
}
