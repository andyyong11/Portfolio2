SportsPro Technical Support Web App

SportsPro is a C# ASP.NET Core MVC web application designed to manage product registrations, customer records, incident tracking, and technician assignments. It was developed collaboratively as part of the CIS296 Advanced C# course.

Features

  • Register customers and assign product support
  • Track incidents and assign technicians
  • Manage support tickets through role-based dashboards
  • Built with ASP.NET Core MVC and Entity Framework Core

Sample Code (from Customer Controller)

                                public IActionResult Register(Customer customer) {
                                    if (ModelState.IsValid) {
                                        context.Customers.Update(customer);
                                        context.SaveChanges();
                                        return RedirectToAction("Complete", customer);
                                    } else {
                                        return View(customer);
                                    }
                                }
